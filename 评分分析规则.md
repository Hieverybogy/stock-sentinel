# Stock Sentinel 评分分析规则文档

## 一、综合评分计算公式

```
综合评分 = 趋势 × 30% + 动量 × 25% + 波动 × 20% + 量能 × 15% + 位置 × 10%
```

每个维度满分100分，最终综合评分为加权总分。

---

## 二、五维评分细则

### 1. 趋势评分（权重30%）

#### 均线排列打分
| 条件 | 得分 |
|------|------|
| MA5 > MA10 | +25分 |
| MA10 > MA20 | +25分 |
| MA20 > MA60 | +25分 |
| 当前价格 > MA5 | +25分 |

#### 均线斜率加分
| 条件 | 得分 |
|------|------|
| MA5斜率为正 | +5分 |
| MA20斜率为正 | +5分 |

**满分100分**，均线多头排列（从上到下依次为 MA5 > MA10 > MA20 > MA60 且价格站上所有均线）为最优。

---

### 2. 动量评分（权重25%）

#### RSI(14日)打分
| RSI区间 | 得分 | 说明 |
|---------|------|------|
| RSI < 30 | +50分 | 超卖，买入机会 |
| 30 ≤ RSI < 40 | +35分 | 偏低 |
| 40 ≤ RSI < 50 | +25分 | 中性偏低 |
| 50 ≤ RSI < 60 | +35分 | 中性偏高 |
| 60 ≤ RSI < 70 | +40分 | 偏强 |
| RSI ≥ 70 | +25分 | 超买，风险积累 |

#### MACD指标
| 条件 | 得分 |
|------|------|
| DIF > DEA（金叉） | +15分 |
| DIF > 0 | +10分 |
| 红柱放大（当前红柱 > 前一红柱） | +15分 |
| 绿柱收缩（当前绿柱 > 前一绿柱的绝对值较小） | +10分 |

#### 短期涨幅
| 5日涨幅区间 | 得分 |
|-------------|------|
| 5% ~ 20% | +10分 |
| 0% ~ 5% | +5分 |

---

### 3. 波动评分（权重20%）

#### ATR波动率
| ATR百分比区间 | 得分 | 说明 |
|---------------|------|------|
| ATR < 1.5% | +40分 | 低波动稳定 |
| 1.5% ≤ ATR < 2.5% | +30分 | 中等波动 |
| 2.5% ≤ ATR < 4% | +20分 | 较高波动 |

#### 布林带状态
| 条件 | 得分 |
|------|------|
| 布林带收口（当前带宽 < 上期带宽 × 0.95） | +30分 |
| 布林带开口（当前带宽 > 上期带宽） | +10分 |

#### 价格在布林带中的位置
| 价格位置区间 | 得分 | 说明 |
|--------------|------|------|
| 20% ~ 80%（中间区域） | +20分 | 正常范围 |
| 40% ~ 60%（中轴附近） | +15分 | 强势区域 |

---

### 4. 量能评分（权重15%）

#### 量比指标
| 量比区间 | 得分 | 说明 |
|----------|------|------|
| 量比 > 1.5 | +40分 | 明显放量 |
| 1.2 < 量比 ≤ 1.5 | +30分 | 温和放量 |
| 0.8 < 量比 ≤ 1.2 | +20分 | 正常量能 |

#### 量价配合
| 条件 | 得分 | 说明 |
|------|------|------|
| 价格上涨且放量（量比 > 1.0） | +20分 | 量价齐升，最强 |
| 价格下跌且缩量（量比 < 1.0） | +15分 | 抛压减轻 |
| 价格上涨但缩量（量比 < 0.8） | +5分 | 量价背离，谨慎 |
| 其他情况 | +10分 | 一般 |

---

### 5. 位置评分（权重10%）

基于60日价格区间的相对位置：

| 相对位置 | 得分 | 说明 |
|----------|------|------|
| < 30% | +50分 | 低位买入区间 |
| 30% ~ 50% | +40分 | 中低位 |
| 50% ~ 70% | +30分 | 中位区 |
| 70% ~ 85% | +20分 | 中高位 |
| ≥ 85% | +10分 | 高位风险区 |

计算方式：`位置 = (当前价格 - 60日最低) / (60日最高 - 60日最低)`

| 0-19分 | 强力卖出 | 红色 | 多指标共振强烈看跌 |

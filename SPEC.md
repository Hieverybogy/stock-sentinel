# StockSentinel - 股票盯盘监测应用规格说明书

## 1. 项目概述

- **项目名称**: StockSentinel (股票哨兵)
- **项目类型**: macOS 本地 Web 应用
- **核心功能**: 实时监控单只股票价格，极速涨跌提醒，5分钟定时报告
- **目标用户**: 短线交易者、需要后台监控的上班族、业余投资者

## 2. UI/UX 规格

### 2.1 窗口布局

- **窗口尺寸**: 最小 420x680px，适合侧边栏放置
- **布局结构**:
  - 顶部标题栏 (macOS 风格)
  - 股票输入区
  - 价格显示区 (主 Hero)
  - 提醒阈值设置区
  - 监控日志区 (可滚动)
  - 底部状态栏

### 2.2 视觉设计

#### 颜色方案 (暗色模式 - 适合盯盘)
- **背景**: `#1C1C1E` (macOS 深灰)
- **卡片/表面**: `#2C2C2E` (浅灰)
- **主要文字**: `#FFFFFF`
- **次要文字**: `#98989D`
- **上涨 (涨)**: `#FF453A` (中国股市标准红色)
- **下跌 (跌)**: `#32D74B` (中国股市标准绿色)
- **强调色**: `#0A84FF` (macOS 蓝)

#### 字体排版
- **字体族**: `-apple-system`, `BlinkMacSystemFont`, `San Francisco`
- **价格显示**: 42px, Bold
- **股票代码**: 16px, Monospace
- **正文**: 14px, Regular
- **日志时间**: 12px, Monospace

#### 视觉效果
- macOS 风格毛玻璃效果
- 价格更新时背景闪烁
- 流畅的过渡动画 (0.3s ease)

### 2.3 组件规格

1. **股票输入框**: 带搜索图标，自动大写，支持回车确认
2. **价格卡片**: 大字显示当前价格和涨跌幅，背景色随涨跌变化
3. **阈值控制**: 极速上涨/下跌触发阈值输入 (默认 2%)
4. **日志列表**: 滚动列表显示带时间戳的监控记录
5. **状态指示**: API 连接状态小圆点

## 3. 功能规格

### 3.1 核心功能

#### 股票数据获取
- 用户输入股票代码 (支持 A 股如 600519.SS，美股如 AAPL)
- 立即获取初始数据
- 使用模拟数据源进行演示 (真实 API 需要密钥)

#### 极速涨跌提醒
- **轮询间隔**: 10 秒 (用于检测极速变化)
- **检测逻辑**: 比较当前价格与 1 分钟前价格
- **触发条件**: `abs(当前价格 - 1分钟前价格) / 1分钟前价格 > 用户阈值`
- **提醒方式**: 
  - macOS 系统桌面通知
  - 页面内视觉闪烁
  - 记录到日志

#### 5分钟定时监控
- **独立计时器**: 每 5 分钟触发一次
- **输出内容**: 
  - 当前时间
  - 当前价格
  - 涨跌幅百分比
- **日志格式**: `[HH:MM] 价格: ¥XXX.XX, 涨幅: +X.XX%`

### 3.2 状态管理

- `currentPrice`: 当前价格
- `priceHistory`: 最近 10 个价格记录 (用于趋势计算)
- `logEntries`: 监控日志条目数组
- `threshold`: 提醒阈值设置
- `stockCode`: 当前监控的股票代码
- `connectionStatus`: API 连接状态

### 3.3 边界情况处理

- **无效股票代码**: 显示错误提示 "未找到该股票"
- **网络断开**: 显示连接失败状态，自动重试
- **市场休市**: 显示 "市场已休市" 状态

## 4. 技术栈

- **框架**: React 18 + Vite
- **样式**: CSS Modules + CSS Variables
- **状态**: React Hooks (useState, useEffect, useRef)
- **通知**: Web Notifications API
- **数据源**: 模拟数据生成器 (开发测试用)
